# SmartRep - AI Fitness Trainer 💪

SmartRep is an AI-powered fitness trainer web app that uses real-time camera pose detection to count reps, detect form mistakes, give voice feedback, and track workout progress.

## 🚀 Features

- Real-time AI pose detection using TensorFlow.js
- AI coaches for Biceps, Squats, Chest, Back, and Abs
- Rep counting and bad-rep detection
- Voice feedback like a fitness coach
- Firebase Authentication with email verification
- Cloud Firestore workout history
- Premium membership system with Razorpay
- Dashboard with progress charts
- Responsive premium UI

## 🛠️ Tech Stack

- HTML
- CSS
- JavaScript
- TensorFlow.js
- Firebase Authentication
- Cloud Firestore
- Razorpay Checkout
- Chart.js
